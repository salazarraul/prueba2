Atributos HTML

No. Atributo Definicion
1 lang Define el lenguaje utilizado en el elemento.
2 manifest Especifica la URL del manifiesto de cache para el documento.
3 usemap Indica la URL parcial de un image map asociado con el elemento.
4 value Define el valor predeterminado a ser mostrado al cargar la página.
5 scoped Indica si el estilo será aplicado sólo al elemento inmediato superior (padre). de no ser definido, el estilo se aplicara de forma global.
6 type Define el tipo de elemento.
7 width esto establece el ancho del elemento.
8 target Especifica el marco destino en un vínculo.
9 start Define el número inicial de la lista (si es diferente a 1).
10 src La URL del contenido integrable.
11 scope Define las celdas que la cabecera definió en este elemento.
12 rel Especifica la relación entre el objeto destino y el objeto enlace.
13 name Nombre del elemento. Por ejemplo utilizado por el servidor para identificar los campos en el envio de formularios.
14 media Da un indicio del tipo de medio para el cual el recurso vinculado fue creado.
15 multiple Indica si múltiples valores pueden ser ingresados.
16 id Identificador el elemento para acceder al mismo desde CSS o Javascript, este indicador debe ser unico para cada elemento.
17 href La URL de un recurso asociado.
18 high Indica el menor valor del rango alto.
19 height Especifica la altura de los elementos enumerados aquí. Para todos los demás elementos, usa la propiedad CSS height en su lugar.
20 reversed Indica si la lista debe ser mostrada en orden descendente en vez de ascendente.
